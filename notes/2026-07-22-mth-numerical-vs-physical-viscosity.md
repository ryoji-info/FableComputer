## 2. The Lax–Friedrichs numerical viscosity (modified-equation, in-model)

`solver._step_LF` is the standard Lax–Friedrichs update `u_j^{n+1} = ½(u_{j+1}+u_{j-1}) − (Δt/2Δx)(f_{j+1}−f_{j-1})`. Its modified equation carries leading numerical diffusion, per characteristic of speed `a`, `D = (Δx²/2Δt)(1 − (aΔt/Δx)²)` — a **frequency-independent (DC)** diffusion, like the DC Navier–Stokes ν₀. In the solver's normalized units (Δx = 1/N, Δt = cfl·Δx/cmax, cmax = 1+M+0.2, DS characteristic speeds a = 1±M):

| N | ν_num (norm, mean ±chars) | ν_num / ν₀ (DC) | released `M_th_num` | offset vs 0.147083 |
|---|---|---|---|---|
| 240 | 6.38×10⁻³ | 0.42 | 0.168943 | +14.86 % |
| 480 | 3.19×10⁻³ | 0.21 | 0.158748 | +7.93 % |
| 720 | 2.13×10⁻³ | 0.14 | 0.155568 | +5.77 % |

with `ν₀(norm) = (v_F²τ_ee/4)/(L·s) = 1.527×10⁻²` (the DC kinematic viscosity, no crossover). `ν_num ∝ 1/N` (first-order, vanishing on refinement — matching the ladder's first-order threshold convergence, demonstrated). **`ν_num(N*) = ν₀` at N* ≈ 100**, so by the momentum-diffusion coefficient the shipped N=240 injects ~0.4× ν₀.

**The gap, flagged (in-model).** The measured N=240 threshold shift (+14.86 %) is the *DC/Navier–Stokes* shift (raw viscous_fraction 0.128, no crossover → ~+15 %, the band edge), larger than the 0.42× momentum coefficient predicts. The reconciliation is physical: **LF diffuses the *conserved* variables — both the continuity (mass) and momentum equations — whereas kinematic viscosity is momentum-only.** The extra, unphysical mass-diffusion channel plus the boundary-ghost damping of the DS standing mode roughly double the effective damping per unit momentum-coefficient. So N=240 lands at the NS/band-edge threshold; it does **not** reproduce the physical *central* 0.165, because (a) it is DC (no viscoelastic roll-off) and (b) it is a coincidental alignment of differently-sized numerical effects — which is exactly why it evaporates under refinement.

## 6. Runnable listing

Reproduces the inviscid anchors, the modified-equation ν_num(N), the promoted crossover-corrected kinetic threshold 0.165, and the tracking cross-check. (The grid ladder and augmented-solver values in §2–§3 come from `solver.growth_rate` runs quoted above.)

```python
# -*- coding: utf-8 -*-
import math, os, sys, json
import numpy as np
CHAIN = r"...\fable-model-chain"                       # adjust to your checkout
sys.path.insert(0, CHAIN)
import constants as C, ds_cell as DS, kinetic as K
RES = json.load(open(os.path.join(CHAIN, "results.json"), encoding="utf-8"))
s = DS.plasmon_speed(); L = DS.cell_length(s); tau = C.tau(C.Tcap); w0 = C.w0; vF = C.vF
Mth = DS.M_threshold(L, s, tau)                        # linearized inviscid = 0.147083
drive = lambda M: ((s**2-(M*s)**2)/(2*s*L))*math.log((s+M*s)/(s-M*s))
def solve_drive(target):
    lo,hi=1e-5,0.5
    for _ in range(200):
        m=0.5*(lo+hi); lo,hi=(m,hi) if drive(m)<target else (lo,m)
    return 0.5*(lo+hi)
Mexact = solve_drive(1.0/(2*tau))
print("M_th linearized = %.6f (json %.6f)" % (Mth, RES["M_th_353K"]))
print("M_th exact inviscid (ds zero) = %.6f (+%.2f%%)" % (Mexact, 100*(Mexact/Mth-1)))
# --- promoted 07-13 kinetic threshold WITH viscoelastic crossover 1/(1+x^2) ---
# (07-13 Appendix B uses the OPERATING length 576.62 nm; its inviscid baseline is 0.147681,
#  0.4% above the ledger's zero-drift small-M M_th_353K=0.147083 -- convention only)
L_op = DS.cell_length(s, M=0.7*Mth)                    # operating length, 576.62 nm
q0=w0/s
def tau_ee(A=1.0): return K.tau_ee(C.Tcap)/A
def x_of(A=1.0): return w0*tau_ee(A)
def gam(M,A=1.0):
    x=x_of(A); nu0=vF**2*tau_ee(A)/4.0
    return 0.5*nu0*q0**2*(1+3*M**2)/(1-M**2)**2/(1+x**2)   # crossover 1/(1+x^2)
def loop(M,A,kin):
    G=1/(2*tau)+(gam(M,A) if kin else 0.0); return (1+M)/(1-M)*math.exp(-G*2*L_op/(s*(1-M**2)))
def Mk(A=1.0,kin=True):
    lo,hi=1e-4,0.45
    for _ in range(200):
        m=0.5*(lo+hi); lo,hi=(m,hi) if loop(m,A,kin)<1 else (lo,m)
    return 0.5*(lo+hi)
Mb=Mk(kin=False)
print("crossover 1/(1+x^2)=%.3f ; eff. viscous_fraction=%.4f (raw %.4f)"
      % (1/(1+x_of()**2), K.viscous_fraction(C.Tcap)/(1+x_of()**2), K.viscous_fraction(C.Tcap)))
print("inviscid baseline (L_op) = %.6f (07-13: 0.147681)" % Mb)
print("M_th_kinetic central A=1 = %.6f (+%.2f%%)  band A=3..0.5: %.4f..%.4f (07-13: 0.165 [0.154,0.169])"
      % (Mk(1.0), 100*(Mk(1.0)/Mb-1), Mk(3.0), Mk(0.5)))
print("shipped M_th_num=%.5f: +%.2f%% vs kinetic central; %.2f%% vs band edge 0.1693"
      % (RES["M_th_num"], 100*(RES["M_th_num"]/Mk(1.0)-1), 100*(RES["M_th_num"]/Mk(0.5)-1)))
# LF numerical viscosity vs DC nu0
nu0 = (vF**2*K.tau_ee(C.Tcap)/4.0)/(L*s)
def nu_num(N, cfl=0.4, M=Mth):
    dx=1/N; cmax=1+abs(M)+0.2; dt=cfl*dx/cmax; pref=dx**2/(2*dt)
    return 0.5*(pref*(1-((1+M)*dt/dx)**2)+pref*(1-((1-M)*dt/dx)**2))
for N in (240,480,720): print("N=%d nu_num/nu0=%.2f"%(N, nu_num(N)/nu0))
print("N* (nu_num=nu0) ~ %.0f" % (240*nu_num(240)/nu0))
# tracking cross-check (raw DC viscous loss, no crossover -> band-edge; -0.95 dB is crossover'd)
d_rt=(nu0*(L*s)*(w0/s)**2/2)*(2*L/s)
print("DC viscous loss/rt = %.2f%% -> tracked %.2f dB (07-13 crossover'd central: -0.95 [-1.5,-0.3])"
      % (100*d_rt, 100*d_rt*-0.284))
```
